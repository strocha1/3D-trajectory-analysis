# 3D-trajectory-analysis
A Python Script used to analyze 3D single-cell trajectories from TrackPy CSV files. 
This code was used to generate the tables used for Figure 4E,F for the following paper: 

Lozano, A. M. et al. Regulatory Plasticity and Metabolic Trade-offs Drive Adaptive Evolution of Alternative Flagellar Configurations in Pseudomonas aeruginosa. 2025.07.29.667523 Preprint at https://doi.org/10.1101/2025.07.29.667523 (2025).
